# TechStore

TechStore é um mini e-commerce desenvolvido em React. O projeto tem como objetivo demonstrar a estrutura inicial de uma aplicação web moderna, com foco em componentização, rotas, organização de código e construção de interface funcional.

## Descrição

A proposta do projeto é simular uma loja virtual simples de produtos de tecnologia, permitindo a navegação entre páginas e a exibição de produtos em uma interface organizada. Nesta primeira etapa, o foco está na base da aplicação, contemplando os requisitos iniciais do projeto da disciplina.

## Objetivos

- Consolidar o uso de React na construção de aplicações modernas.
- Aplicar conceitos de componentização e rotas.
- Desenvolver uma interface funcional e bem estruturada.

## Tecnologias utilizadas

- React.
- Vite.
- React Router DOM.
- JavaScript.
- CSS.

## Estrutura do projeto

O projeto foi organizado em pastas para facilitar a separação de responsabilidades e a manutenção do código, seguindo a proposta do enunciado de usar `components`, `pages`, `services` e outras divisões equivalentes.

```bash
src/
  components/
  pages/
  data/
  styles/
```

## Funcionalidades implementadas

- Estrutura inicial da aplicação com React.
- Organização do projeto em pastas.
- Criação de componentes reutilizáveis.
- Uso de props para renderização dos produtos.
- Implementação de rotas com React Router.
- Múltiplas páginas, incluindo Home, Produtos e Sobre.
- Layout padrão com header, menu de navegação e footer.
- Exibição dinâmica de produtos a partir de dados locais.

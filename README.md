# Y.com

Y é uma rede social simples inspirada no X/Twitter.
Inicialmente criado com intuito educacional. (Projeto Web Developments Frameworks - Sistemas de Informação PUCPR 3° Período)

O usuário pode fazer login, criar posts, comentar, responder comentários e enviar imagens ou GIFs.

## Tecnologias

- JavaScript
- HTML/CSS
- React
- React Router
- API de autenticação com JWT
- fetch para requisições HTTP

## Funcionalidades

- Login e cadastro
- Autenticação com JWT
- Feed de posts
- Comentários e respostas
- Upload de imagem/GIF
- Página de perfil

## Como rodar

```bash
npm install
npm run dev
```
